# SpringMVP Project

A Spring Boot MVC project template.

## Project Structure

```
springmvp/
├── src/
│   ├── main/
│   │   ├── java/com/ai/mvp/
│   │   │   └── SpringmvpApplication.java
│   │   └── resources/
│   │       └── application.properties
│   └── test/
│       └── java/com/ai/mvp/
│           └── SpringmvpApplicationTests.java
├── pom.xml
├── mvnw
└── README.md
```

## Getting Started

### Prerequisites
- Java 17+
- Maven 3.6+

### Running the Application
```bash
./mvnw spring-boot:run
```

### Building
```bash
./mvnw clean package
```

## Features
- Spring Boot 3.x
- MVC architecture
- Maven wrapper
